# Smart Workforce Analytics & Management System Presentation Guide

## Team Roles

### Member 1
- Introduce the project
- Explain the problem statement
- Explain the objective

### Member 2
- Explain technologies used
- Explain system architecture
- Explain OOP classes and features

### Member 3
- Give demo
- Explain outputs and charts
- Present conclusion and future scope

## Simple Explanation

We built a Python-based workforce management system that stores employee data in SQLite, analyzes it using Pandas and NumPy, creates charts with Matplotlib, and presents everything through a Flask web app.
